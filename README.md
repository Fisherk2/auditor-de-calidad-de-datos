# Plantilla para Proyectos de Ciencia de Datos

## Propósito
Esta plantilla proporciona una estructura de directorios y archivos base para iniciar proyectos de ciencia de datos de manera organizada y profesional.

## Cómo Usar
1. Haz clic en "Use this template" en GitHub.
2. Asigna un nombre a tu nuevo repositorio.
3. Clona el repositorio recién creado.
4. Personaliza esta documentación y comienza a trabajar.

## Convenciones de Nombrado
- Carpetas: `snake_case`
- Notebooks: `nombre_fase_descripcion.ipynb` (ej. `01_exploracion_datos.ipynb`)
- Scripts: `modulo_nombre_funcion.py` (ej. `data_load_data.py`)

## Ejemplo de Flujo de Trabajo
1. Explorar datos en `notebooks/exploratory`.
2. Preparar datos en `src/data/`.
3. Extraer características en `src/features/`.
4. Entrenar modelos en `src/models/`.
5. Visualizar resultados en `src/visualization/`.
6. Guardar resultados en `results/`.
